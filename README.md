# Contact Travel

Our `contact_travel` module.


## Tested with

- Odoo version 17
- PostgreSQL 16
- Windows 10 Pro x64


## Running locally

```cmd
venv\Scripts\activate

python odoo\odoo-bin -d contact_travel -c conf\odoo.conf -i contact_travel
```

Visit http://localhost:8069/

Installing: Search for "Contact Travel" and **activate** it.
